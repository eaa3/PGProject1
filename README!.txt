Instruções:

teclas w,s,a,d transladam a câmera
teclas up,left,right,down,q,e rotacionam a câmera
teclas y,g,h,j,t,u rotacionam o objeto
teclas f,v,c,b,z,x transladam o objeto
tecla space aplica transfomação de escala no objeto
tecla n aplica transformação de contração no objeto
tecla tab seleciona um objeto diferente
tecla esc fecha o programa
